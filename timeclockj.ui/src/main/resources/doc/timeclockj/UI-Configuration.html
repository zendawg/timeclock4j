4.1 UI Configuration

Several command-line options are available when starting the UI. As well as the options given in Common Command Line Options, the following options are available:

  • -i,--interval <arg> Set time (seconds) for time to wait before checking if file has changed. Default is 1. Use with --reload-on-change
  • -d,--report-dir <arg> Use the specified directory to write reports to; will be created if it does not exist. Defaults to the user’s home directory in a file named ‘.timeclockj/reports’
  • -r,--reload-on-change If file has changed, reload. Use with --interval
