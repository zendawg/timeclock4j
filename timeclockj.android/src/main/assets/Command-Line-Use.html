3. Command Line Use

The timeclockj tool is used for manipulating timeclock files. The basic syntax for command-line use is:

 
timeclockj [command] [options]

…where [command] is one of in (for clocking in), out (for clocking out) or report for generating reports.

Supplying the command is optional - see Common Command Line Options for command-line options that do not require any a command to be executed.
