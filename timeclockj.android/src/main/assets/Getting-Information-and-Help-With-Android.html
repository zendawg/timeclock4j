5.4 Getting Information and Help With Android

The information tab gives useful information about the version, license, contributors, help pages and file data:

The different functions of the buttons are as follows:

  • About button, detailing the version of timeclockj being used, as well as license information and project website;
  • Contributors button, detailing different contributors to the project;
  • Help button - selecting this will launch the embedded help pages;
  • File data information, such as last modification date.
