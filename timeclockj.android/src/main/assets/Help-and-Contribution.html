7. Help and Contribution

timeclockj is in it’s first alpha release; there are probably many bugs and issues that need resolving.

Please visit https://sourceforge.net/projects/timeclockj/support for any issues to do with feature requests, bug reports or other issues.

If you’d like to get involved with timeclockj on any level (development, user testing, documentation), feel free to contact the timeclockj team at Source Forge, https://sourceforge.net/projects/timeclockj.
