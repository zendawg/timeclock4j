6.1 GNU Emacs Syncronisation

All you need to do with GNU emacs is set the file it uses to clock times to a point to a Dropbox directory containing your timeclock file. How to change the file name variable is already documented at the following URL:

http://www.delorie.com/gnu/docs/emacs/emacs_437.html
